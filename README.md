## DESCRIPTION

The purpose of this repo is to provide a syncable bundle of fonts.\

> Implemented as separate git-repos (included as submodules).

- [mono](#mono)
- [sans](#sans)
- [serif](#serif)
- [emoji](#emoji)
- [misc](#misc)
